## 3. 초기화 로직 (중요)

단순히 "자정에 리셋"이 아니라 **06:00 기준**이라는 점이 핵심. 즉, 새벽 2시에 앱을 열면 "어제치" 체크 상태가 유지되어야 함.

### 기준일 계산

```
현재 시각이 오늘 06:00 이전  → 기준일 = 어제 (YYYY-MM-DD)
현재 시각이 오늘 06:00 이후  → 기준일 = 오늘 (YYYY-MM-DD)
```

### 저장 스키마 (제안)

```json
{
  "baselineDate": "2026-04-23",
  "checked": {
    "lostark_daily_chaos": true,
    "lostark_daily_guardian": false,
    "maple_daily_boss": true
  }
}
```

페이지 로드 시 저장된 `baselineDate` ≠ 현재 계산된 기준일이면 `checked`를 `{}`로 리셋하고 `baselineDate` 갱신.
